# 一、if-elif-else 条件语句

### 1. 基本语法
```python
if 条件1:
    代码块1
elif 条件2:
    代码块2
else:
    代码块3
```

### 2. 如何使用？
- `if`   条件满足时执行对应代码
- `else`表示所有条件都不满足时执行的代码块
- `elif`   用于判断多个条件分支，即条件1+...+条件n满足时要执行的代码块

# 二、while循环

### 1. 基本语法
```python
while 条件:
    代码块
```

### 2. 如何使用？
- 在条件为True时循环执行代码块，直到条件为False

### 3.循环控制
- `break`   直接终止循环
- `continue`   终止当前一轮的循环
<mark style="background:rgba(240, 107, 5, 0.2)">！！！如果有多个循环，则`break`只会终止对应的一个循环</mark>

---
# 三、for循环

### 1. 基本语法
```python
for 变量 in 数据集:
    代码块
```

### 2. 如何使用？
- 常用于遍历列表、元组、字符串、字典等可迭代对象。
- 每次循环将元素赋给变量，执行代码块。
- 可配合`break`和`continue`控制循环。

### 3. range( )
- 基本语法：
```python
range(起始值，终止值，步长)
```

- ==**包左不包右**==
- 如果以0为起始值，则可以写成 `range(40)`

### 4. 注意
- 如果迭代对象为空，循环体不会被执行

# 四、异常捕获（try-except）

### 1. 基本语法
```python
try:
    可能出错的代码
except 异常类型:
    出错时执行的代码
```

### 2. 捕获指定异常
- 基本语法：
```python
try:
	可能出错的代码
except 指定的错误:
    print("~~错误")
```
- 如果出现的异常类型与要捕获的异常类型不一致，则无法捕获异常

### 3. 处理多种异常
- 可以用多个`except`，捕获不同类型的异常：
```python
try:
    可能出错的代码
except ValueError:
    print("值错误")
except TypeError:
    print("类型错误")
```
- 还可以用==元组==同时捕获多种异常：
```python
try:
    可能出错的代码
except (ValueError, TypeError):
    print("出现值错误或类型错误")
```

### 捕获全部异常
```python
try:
	可能出错的代码
except Exception:
    出错时执行的代码
```
- 其实常规的捕获异常就是捕获全部异常了，这一个只是刷个脸熟

### 4. 如何捕获异常对象？
- 我们用 `as` 获取错误对象的详细信息：
```python
try:
    可能出错的代码
except ValueError as e:
    print("异常信息：", e)
```

### 5. else 和 finally
- `else`：只有==没有异常==时才会执行
- `finally`：无论是否出现异常都会执行，一般用于资源清理，释放资源清理环境

# 五、列表推导式

### 1. 基本语法
```python
[表达式 for 变量 in 可迭代对象 if 条件]
```

- 前面的“表达式”一般是你希望的新列表元素，可以是变量、运算甚至函数调用等
- “变量”是每轮遍历取出的元素
- “可迭代对象”比如列表、元组、字符串、集合等
- “if 条件”是可选项，只有满足条件才加入新列表

### 2. 举个栗子
```python
# 生成1~5的平方列表
squares = [x ** 2 for x in range(1, 6)]

# 取出偶数
evens = [x for x in range(10) if x % 2 == 0]

# 字符串处理：去除空行
lines = [line.strip() for line in lines if line.strip() != ""]
```

- **集合推导式示例：**
```python
{表达式 for 变量 in 可迭代对象 if 条件}
```

- **字典推导式示例：**
```python
{键表达式:值表达式 for 变量 in 可迭代对象 if 条件}
```

# 二、函数式编程核心语法
## 1.什么是函数式编程？
- 把函数当做变量传递
- 把函数作为参数、返回值
- 使用“纯函数”来使用，即没有副作用的函数
- 不直接修改数据，而是生成新数据

## 2. 函数式编程工具 
### 1. lambda表达式（匿名函数）
- **基本语法：**
```python
lambda 参数1, 参数2, ...: 表达式
```

- lambda用于快速定义临时、简单函数，==没有函数名==
- lambda的参数和表达式只能写在==一行==里，所以不能有复杂语句
- **举个栗子**
```python
f = lambda x: x ** 2
print(f(3))  # 输出 9
```

### 2. map 函数
- **基本语法：**
```python
map(函数, 可迭代对象)
```

- 将函数应用到可迭代对象的每一个元素，返回新的迭代器
- map返回的是==迭代器==，不是列表，所以需要用`list()`、`tuple()`转换为具体的类型
- **举个栗子**
```python
# 所有元素取平方
list(map(lambda x: x ** 2, range(1, 6)))
```

### 3. filter 函数
- **基本语法：**
```python
filter(函数, 可迭代对象)
```

- 把函数作用于每个元素，只有返回True的元素才保留
- 同样的，filter函数返回的是==迭代器==，所以也需要用`list()`、`tuple()`转换为具体的类型
- **举个栗子**
```python
# 只保留偶数
list(filter(lambda x: x % 2 == 0, range(1, 6)))
```

### 4. reduce 函数
- **基本语法：**
```python
reduce(函数, 可迭代对象[, 初始值])
```
- 用于累积处理序列元素，适合聚合操作（如求和、连乘、拼接字符串等）
- 第一次用时需要导入functools模块：
```python
from functools import reduce
```

- **举个栗子**
```python
# 求列表所有元素的和
reduce(lambda x, y: x + y, [1, 2, 3, 4, 5])
```
