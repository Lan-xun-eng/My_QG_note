# 循环

> Python中，循环主要包含：while循环 和 for循环

## while循环
- **格式：**
```
while 条件：
	循环体
```

## for循环

- **格式：**
```
for 变量 in 数据集：
	循环体
```
- 在for循环中，变量名是任意的，然而，选择描述单个列表元素的变量名会更好
  即`for name in names:`

## 循环控制：

- break：直接终止循环
- contiue：只终止当前一轮循环

<mark style="background:rgba(240, 107, 5, 0.2)">！！！注意： 如果有多个循环，break只是终止对应级的循环，而非全部终止</mark>
