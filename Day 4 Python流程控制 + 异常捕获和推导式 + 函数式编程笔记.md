# 一、if-elif-else 条件语句

### 1. 基本语法
```python
if 条件1:
    代码块1
elif 条件2:
    代码块2
else:
    代码块3
```

### 2. 如何使用？
- `if`   条件满足时执行对应代码
- `else`表示所有条件都不满足时执行的代码块
- `elif`   用于判断多个条件分支，即条件1+...+条件n满足时要执行的代码块

# 二、while循环

### 1. 基本语法
```python
while 条件:
    代码块
```

### 2. 如何使用？
- 在条件为True时循环执行代码块，直到条件为False

### 3.循环控制
- `break`   直接终止循环
- `continue`   终止当前一轮的循环
<mark style="background:rgba(240, 107, 5, 0.2)">！！！如果有多个循环，则`break`只会终止对应的一个循环</mark>

# 三、for循环

### 1. 基本语法
```python
for 变量 in 数据集:
    代码块
```

### 2. 如何使用？
- 常用于遍历列表、元组、字符串、字典等可迭代对象。
- 每次循环将元素赋给变量，执行代码块。
- 可配合`break`和`continue`控制循环。

### 3. range( )
- ==**包左不包右**==
- 以0为起始值  ->  range(40)
- 常常用于for循环遍历

### 4. 注意
- 如果迭代对象为空，循环体不会被执行。

# 四、异常捕获（try-except）

### 1. 基本语法
```python
try:
    可能出错的代码
except 异常类型:
    出错时执行的代码
```

### 2. 处理多种异常
- 可以用多个 `except`，捕获不同类型的异常：
```python
try:
    可能出错的代码
except ValueError:
    print("值错误")
except TypeError:
    print("类型错误")
```
- 还可以用元组同时捕获多种异常：
```python
try:
    可能出错的代码
except (ValueError, TypeError):
    print("出现值错误或类型错误")
```


> [!NOTE] 札记
> foo()、bar()等函数是常见的举例函数

### 3. 如何捕获异常对象？
- 我们用 `as` 获取错误对象的详细信息：
```python
try:
    可能出错的代码
except ValueError as e:
    print("异常信息：", e)
```

### 4. else 和 finally
- `else`：只有==没有异常==时才会执行
- `finally`：无论是否出现异常都会执行，一般用于资源清理，释放资源清理环境

###

### 2. 使用方法
- `try`用于捕获代码块中的异常。
- `except`用于捕获并处理指定类型的异常。
- 可多个`except`分支处理不同类型异常。
- `else`块在没有异常时执行。
- `finally`块无论是否异常都会执行，如资源释放。

### 3. 常见注意事项
- 尽量捕获具体异常类型，避免只用`except`。
- 捕获异常对象可用`as e`引用异常实例。
- 保证`finally`释放资源清理环境。
- 不要滥用异常捕获，先用条件判断���止异常。
