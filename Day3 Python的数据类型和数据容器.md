# 数据容器
## 列表
- **列表名 = [ 元素1，元素2，元素3，~ ]**
- 列表可以加入**任何东西**，其中的元素之间可以**没有任何关联**

### 访问列表元素：
- 列表名 [ 索引 ]
- 索引是从 0 开始的
- ==索引 = -1== ： 直接返回列表==最后一个元素==
	- 同理，索引=-2 → 返回列表倒数第二个元素

### 修改元素：
- 列表名[ 索引 ] = 元素

### 添加元素：
- **==append( 元素 )==**   ：   从列表末尾添加元素
- **==列表名.insert[ 索引 , 元素 ]==**   ：   在列表中插入元素，在任何位置添加新元素

### 删除元素：
- **==del( 索引 )==**   →   删除列表中任意位置的元素
- **==pop( )==**   →   删除列表末尾的元素+接着使用它 → 在pop( 索引 )加入索引，则可以删除并弹出列表中的任意元素
- **==remove( 元素 )==**  → 删除列表中某一元素的值
	- remove( )只删除第一个指定的值，所以如果列表中有多个这样的值并要全部删除，则需要使用循环
#### 列表元素删除方法的选择：
- 知道元素的位置 → 用del()
- 知道要删除元素的值 → 用remove()
- 知道位置，但想在删除元素后还能继续使用它 → 用pop()

### 列表相关方法：
- 永久排序：用方法sort()
- 临时排序：用函数sorted()
- 倒着打印列表 → 用方法reverse()
- 确定列表长度 → 用函数len()

### range( )
- **range(起始，终止)**
- range( )是从起始值开始，到终止值-1结束
- range(起始，终止，步长)
- 用range()创建数字列表：列表名 = list(range(~,~))

### 切片：
- **列表名[~：~]**  
- 切片从**起始值**开始，至**终止值-1**结束 → 与range()相同  
- 复制列表，就要使用切片，相当于创建一个列表副本 但是，复制列表**绝不可以**直接将列表2 = 列表1

# 集合：

- **集合名 = { 元素1，元素2 }**
- 与字典的主要区别在于：字典的{}中是**键值对**，而集合的{}中是**元素即值**
- 集合中每个元素都是独一无二的，即集合中的元素**不能重复**

# 字典

- **字典名 = { 键1：值1 ，键2：值2 ，键3：值3 }**  
    
- **字典名 = { } / 字典名 = dict()** → 定义空字典  
    
- 字典中的键和值可以是任何类型  
    

### 访问字典中的值：

- **字典名[ 键 ]**  
    
- 但是，如果指定的键不存在则会报错  
    
- **get( 键 ，指定的键不存在时要返回的值 )**  
    
- 选择： 如果指定的键如果有可能不存在则使用get() 一般都用字典名[ 键 ]  
    

### 添加键值对：

- **字典名[ 新的键 ] = 新的值**
- 新添加的键值对直接接在字典的末尾

### 修改字典中的值：

- **字典名[ 键 ] = 新的值**

### 删除键值对：

- 2种方法：
    - **del 字典名[ 要删除的键 ]**
    - **pop( )**

### 清空字典：

- clear( )

### 遍历字典：

- 遍历字典时，一定要先将**字典**转换成**列表** → **字典名.items()**
- 注意是**items()**

```python
user_0 = {	'username':'efermi',	'first':'enrico',	'last':'fermi',	}for key,value in user_0.items():	print(key)	print(value)
```

### 遍历字典中的所有键/值：

- **字典名.keys()** → 遍历字典中的所有键  
    
- **sorted(字典名.keys())** → 按特定顺序遍历字典中的所有值  
    
- **字典名.values()** → 遍历字典中的所有值  
    
- 同样的，注意是**keys() ， values()**  
    
- **set( 字典名.values() )** → 去除字典中值的重复项 → 本质上就是集合
